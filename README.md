# Tech Tycoon 💼

A beginner-friendly tycoon game built using Python where players manage a business and grow their income.

## Features
- Earn money
- Upgrade business
- Increase income over time
- Simple interactive game loop

## How to Run
1. Install Python
2. Run: python tycoon.py
